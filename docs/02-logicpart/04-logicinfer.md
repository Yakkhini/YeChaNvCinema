# 基本推断

在“逻辑核心”一章中，我们阐述了逻辑最基本的模式，即xx是一种xx，或xx非一种xx。但问题在于，这样的模式仅仅是一种静态的描述，而不包含有一种具有连贯性的，动态的交互逻辑。我们在这一节里将进一步展示在这一逻辑体系中逻辑推断的表现方式，并把握其去中心的特性，理解“事物本身并不重要”的意义。

## 经典三段推论与经典符号学工作的先验中心

> 所有动物都会死。
>
> 所有人都是动物。
>
> 所以，所有人都会死。
